# **App Name**: Sentinel Scan

## Core Features:

- Link Extraction: Automatically extract URLs from text messages.
- Malicious Link Detection: Analyze extracted URLs against a database of known malicious links.
- Risk Assessment: Score the risk level of each link.
- Community sharing and ratings: Rate scanned links to enrich the database of known malicious links, to support others.
- Link safety feedback: Warn users when the link is a scam and offer clear steps for recourse and support. Tell user what to do if they already clicked that link?
- History storage: Keep history of scanned links to monitor links over time

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a modern and secure feel.
- Background color: Dark blue (#1A237E) with low saturation to maintain a dark theme.
- Accent color: Teal (#26A69A) to highlight safe links and interactive elements.
- Font: 'Inter' sans-serif for body and headline as it's a clean, modern font suitable for UI.
- Use simple, geometric icons for navigation (history, community), safety ratings.
- Implement a dark mode interface with glassmorphism-inspired frosted glass effect for information cards and input fields.
- Add subtle animations on link scanning, background movement, cursor interactions, and status changes (safe/unsafe).
